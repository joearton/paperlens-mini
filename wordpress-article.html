üì¶ Installation & Usage

Quick Start (3 cara):

1. Download Executable (Paling Mudah)

  • Windows: Sintesa.exe (~500 MB)
  • macOS: Sintesa.app (~500 MB)
  • Linux: Sintesa (~500 MB)

Double-click dan langsung jalan, no installation required!

2. Run from Source (Developer)

cd sintesa
pip install -r requirements.txt
python app.py

3. Use Run Scripts

  • Windows: run.bat
  • Linux/Mac: ./run.sh

Auto-setup virtual environment dan install dependencies.
